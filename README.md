# ClauseWise Leguism

## Problem
Terms and conditions are lengthy and difficult to understand. This project summarizes complex legal text into simple and understandable content using NLP and transformer models.

## Tech Stack
- Python
- spaCy
- T5 Transformer (Fine-tuned)
- Streamlit
- Azure Speech SDK

## Features
- Text summarization using transformer models
- NLP-based text processing
- Text-to-speech support for accessibility
- Interactive web interface using Streamlit

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the Streamlit app:
   streamlit run app.py
